JavaScript sort() 함수 - 화살표 함수 사용

sort() 메서드는 비교 함수 (a, b)를 받아 정렬 순서를 결정합니다.

return 1이면 a가 b보다 크므로 자리 변경 (오름차순 정렬).

return -1이면 a가 b보다 작으므로 순서 유지.

예제 코드

const arr = [2, 1, 3, 10];

// 일반 함수 사용
arr.sort(function(a, b) {
    return a - b;
});

// 화살표 함수 사용 (간단하게)
arr.sort((a, b) => a - b);

console.log(arr);  // [1, 2, 3, 10]

정렬 과정

[2, 1, 3, 10]

  1. 첫 번째 비교 (2와 1) → 작은 숫자와 자리 바꾸기 → [1, 2, 3, 10]
  2. 두 번째 비교 (2와 3) → 작은 숫자 찾기 → 순서 유지 → [1, 2, 3, 10]
  3. 세 번째 비교 (3와 10) → 작은 숫자 찾기 → 순서 유지 → [1, 2, 3, 10]
