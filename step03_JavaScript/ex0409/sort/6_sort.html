JavaScript sort() 함수 - 문자열 내림차순 정렬

문자열 내림차순으로 정렬하기

내림차순으로 정렬하기 위해 비교 함수(compareFunction)를 사용하여 정렬합니다.

예제 코드

const arr = ['banana', 'b', 'boy'];

arr.sort(function(a, b) {
    if (a < b) return 1;
    if (a > b) return -1;
    if (a === b) return 0;
});

console.log(arr);  // ['boy', 'banana', 'b']
