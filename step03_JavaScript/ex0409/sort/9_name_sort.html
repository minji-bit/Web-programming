JavaScript sort() 함수 - 객체의 name 값 오름차순 정렬

객체의 name 값을 기준으로 오름차순 정렬

객체 배열에서 문자열 속성(name)을 기준으로 정렬할 때 화살표 함수를 사용합니다.

예제 코드

const arr = [
    {name: 'banana', price: 3000}, 
    {name: 'apple', price: 1000},
    {name: 'orange', price: 500}
];

arr.sort((a, b) => a.name < b.name ? -1 : a.name > b.name ? 1 : 0);

console.log(arr);  
// 0: {name: 'apple', price: 1000}
// 1: {name: 'banana', price: 3000}
// 2: {name: 'orange', price: 500}
