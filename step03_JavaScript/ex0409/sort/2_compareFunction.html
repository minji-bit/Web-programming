JavaScript sort() 함수 - compareFunction

sort() 함수는 기본적으로 유니코드 순서에 따라 정렬합니다.

예를 들어, [2, 1, 3, 10] 배열을 정렬하면 [1, 10, 2, 3]이 됩니다.

이유: 숫자를 문자열로 변환하여 유니코드 순서로 정렬하기 때문에 "10"이 "2"보다 먼저 옵니다.

기본 정렬 예제

const arr = [2, 1, 3, 10];
arr.sort();
console.log(arr);  // [1, 10, 2, 3]

정확한 숫자 정렬을 위한 compareFunction 사용

오름차순으로 정렬하기 위해 compareFunction을 사용합니다.

const arr = [2, 1, 3, 10];
arr.sort(function(a, b) {
    return a - b;
});
console.log(arr);  // [1, 2, 3, 10]
